1. Unity介绍

4.4 五个窗口面板的作用

在Unity中，有五个主要的面板

  1. Scene 场景面板，显示当前场景的样子

    鼠标中键：平移视角

    鼠标右键：旋转视角（环顾四周），围绕自身旋转

    Alt+鼠标左键：渲染视角，围绕当前视角前方位置

  2. Game 游戏面板，场景运行的时候的样子

  3. Hierarchy 层级面板

    • 展示当前场景Scene有哪些游戏物体

    • 会根据点击不同的场景Scene进行转换

    • Main Camera 相机

    • Directional light 太阳

  4. Project 工程面板，存放工程的各种资源

    • Assets 存放资源
    • Packages 保存插件

  5. Inspector 检视面板（也可以叫属性面板）

    • 展示当前游戏物体GameObject有哪些组件Component

4.12 通过刚体控制小球运动

可以在脚本中添加代码如下

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class NewBehaviourScript : MonoBehaviour
{
    public Rigidbody Rd { get; set; }	//如果这么写在Unity中不会显示Rd,去掉后面的getset后才显示，不过不影响运行结果
    // Start is called before the first frame update
    void Start()
    {
        Rd = GetComponent<Rigidbody>(); //获取组件
    }

    // Update is called once per frame
    void Update()
    {
        Rd.AddForce(Vector3.right); //添加力，Vector3指三维向量，向左施加一个1N的力
    }
}

回到Unity运行之后小球就会一直向右移动

方法总结

第一种方法 - 通过代码获取

private Rigidbody rd;

rd = GetComponent<Rigidbody>();

第二种方法 - 通过拖拽（不推荐）

public Rigidbody rd;

给小球添加力

rd.AddForce(Vector3.right);	//向右1N
rd.AddForce(new Vector3(5,0,0));	//向右5N

知识点1：方法调用

x.AddForce 也可以叫做调用指令，调用指令的时候需要通过（）传递参数。

小明你去买点菠菜。 小明.买(菠菜) 小明.买菠菜()

知识点2：三维向量 (x,y,z)

Vector3.forward 等于 （0,0,1）

向量两个要素（方向，长度） （0,0,1） （0,0,4）

知识点3：一些常用的向量

Vector3.right Vector3.left Vector3.forward Vector3.back Vector3.up Vector3.down

知识点4：怎么创建向量

new Vector3(x,y,z)

通过这种方法小球的运动是慢慢加快的，就和现实中一样，一个物体受到一个力由静止做加速运动

4.13 监听键盘并控制小球运动

首先我们在Update()方法中添加如下两行代码

void Update()
{
    float h = Input.GetAxis("Horizontal");
    Debug.Log(h);
}

然后回到Unity进行监听，默认为0

尝试按A键并保持，我们发现当我们按A键时，不是立即输出-1，而是从0过度到-1，保持之后会一直在-1，松开之后会从-1过度到0

如果我们按D则相反为1

接下来我们修改代码如下

回到Unity运行，我们发现小球可以根据AD键进行左右移动了

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class NewBehaviourScript : MonoBehaviour
{
    public Rigidbody Rd;
    // Start is called before the first frame update
    void Start()
    {
        Rd = GetComponent<Rigidbody>(); //获取组件
    }

    // Update is called once per frame
    void Update()
    {    
        float h = Input.GetAxis("Horizontal"); //A-1 D1
        float v = Input.GetAxis("Vertical"); //WS
        Debug.Log(h);   
        Rd.AddForce(new Vector3(h, 0, v));
    }
}

如上代码我们就可以通过WASD和方向键共8个键控制小球移动了

4.14 控制相机跟随移动

让小球刚好贴到体面上，且位于中心只要设置**XYZ(0,0.5,0)**，

Y是上下，0.5是小球的半径，意思就是小球的中心点向上半径的位置

将Main Camera设置如下，主要用到了WE工具

然后在Main Camera添加一个C#脚本，名为FollowTarget，将Plyaer通过拖拽添加进去

FollowTarget代码如下

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class FollowTarget : MonoBehaviour
{
    public Transform playerTransform;
    private Vector3 offset;
    // Start is called before the first frame update
    void Start()
    {
        offset = transform.position - playerTransform.position; //计算小球与相机的偏移量
    }

    // Update is called once per frame
    void Update()
    {
        transform.position = playerTransform.position + offset; //设计相机的位置
    }
}
