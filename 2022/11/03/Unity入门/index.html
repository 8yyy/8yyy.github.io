1. Unity介绍

​ Unity 是实时3D互动内容创作和运营平台 。包括游戏开发、美术、建筑、汽车设计、影视在内的所有创作者，借助Unity将创意变成现实。 Unity平台提供一整套完善的软件解决方案，可用于创作、运营和变现任何实时互动的2D和3D内容，支持平台包括手机、平板电脑、PC、游戏主机、增强现实和虚拟现实设备。

​ Unity 引擎的灵活性使开发者能够为超过20个平台创作和优化内容 ，这些平台包括iOS、安卓、Windows 、[Mac OS](https://baike.baidu.com/item/Mac OS/2840867?fromModule=lemma_inlink)、索尼PS4、任天堂Switch 、微软[Xbox One](https://baike.baidu.com/item/Xbox One/4728387?fromModule=lemma_inlink)、谷歌Stadia、微软Hololens、谷歌AR Core、苹果AR Kit、商汤SenseAR等等 。公司超过1800人 规模的研发团队，同时跟随合作伙伴迭代 ，确保在最新的版本和平台上提供优化支持服务。

​ 不仅提供创作工具 ，Unity还提供运营服务来帮助创作者 。这些解决方案包括：Unity Ads广告服务、Unity游戏云一站式联网游戏服务、Vivox游戏语音服务、Multiplay海外服务器托管服务 、Unity内容分发平台（UDP）、Unity Asset Store资源商店 、Unity云构建等。

​ Unity总部位于美国加利福尼亚州旧金山，并在丹麦、比利时、立陶宛、哥伦比亚 、加拿大、中国、芬兰、瑞典、德国、法国、日本、英国、爱尔兰、韩国和新加坡设有办公室。 创作者分布在全球190个国家和地区。

​ ———摘自百度百科

一般最好下载LTS（Long Term Support）长期支持版本

对了，因为Unity和腾讯有合作，所以可以直接使用微信登录Unity官网

然后我这个Unity Hub是之前下载VS的时候一起安装了的，然后点击安装发现容量不够，所以我选择自己安装，回到官网，选择2021.3.12版本，点击下载，当然你也可以选择更新的版本

4. 第一个Unity游戏RollABall

SIKI学院的一个实例

4.1 新建项目

首先需要在设置里激活许可证，选择个人版就好

到项目面本，点击新建一个3D游戏RollABall，我这里拼错了

创建完后之后打开是这样的，默认是Default，可点击右上角进行修改

4.2 认识工程目录

  1. Assets 资源，存放素材脚本场景等等
  2. Library 库
  3. Logs 日志
  4. Packages 包
  5. ProjectSettings 设置
  6. Temp 临时文件夹
  7. UserSettings

然后我发现没有sln文件，经过搜索得知

这个问题主要是没有在 edit -> preferences -> External Tools -> external script editor中设置你需要的启动器所以根本没生成sln文件，只要把这个设置为你使用的VS版本就可以了

4.3 如何创建代码和场景

目前Unity脚本只支持C#,放在Assets文件夹下，过程如下

右键Assets文件夹 –> create –> c# scripts

就会创建一个名为NewBehaviourScript的C#脚本，可以给他重命名下，双击脚本，即可打开VS

打开VS默认如下，发现之前做坦克大战的时候差不多的两个方法

打开工程文件夹，就可以看到刚才创建的脚本文件

Scenes是指游戏场景，我们可以在Unity当中

右键Assets下的Scenes文件夹 –> create –> Scene 创建游戏场景，默认会存在一个SampleScene场景

4.4 五个窗口面板的作用

在Unity中，有五个主要的面板

  1. Scene 场景面板，显示当前场景的样子

    鼠标中键：平移视角

    鼠标右键：旋转视角（环顾四周），围绕自身旋转

    Alt+鼠标左键：渲染视角，围绕当前视角前方位置

  2. Game 游戏面板，场景运行的时候的样子

  3. Hierarchy 层级面板

    • 展示当前场景Scene有哪些游戏物体

    • 会根据点击不同的场景Scene进行转换

    • Main Camera 相机

    • Directional light 太阳

  4. Project 工程面板，存放工程的各种资源

    • Assets 存放资源
    • Packages 保存插件

  5. Inspector 检视面板（也可以叫属性面板）

    • 展示当前游戏物体GameObject有哪些组件Component

一个场景包含多个游戏物体，每个游戏物体包含多个组件
