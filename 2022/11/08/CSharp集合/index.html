1. 前言

用过C++ STL容器的都知道STL容器非常好用，其封装了一些常用的数据结构，很方便使用。那么在C#中是否有类似与STL容器的东西呢？是有的，在C#集合(Collections)中，例如在STL中，最常用的容器要算vector、list、map、set这四种了。C#中，对应的容器分别是:List、LinkedList、Dictionary、HashSet。

2. 集合的类型

集合是一个类，因此必须在向该集合添加元素之前，声明类的实例。

如果集合中只包含一种数据类型的元素，则可以使用 System.Collections.Generic 命名空间中的一个类。 泛型集合强制类型安全，因此无法向其添加任何其他数据类型。 当你从泛型集合检索元素时，你无需确定其数据类型或对其进行转换。

集合有三个重要的类型：

  • System.Collections.Generic 类
  • System.Collections.Concurrent 类
  • System.Collections 类

2.1 System.Collections.Generic 类

可以使用 System.Collections.Generic 命名空间中的某个类来创建泛型集合。 当集合中的所有项都具有相同的数据类型时，泛型集合会非常有用。 泛型集合通过仅允许添加所需的数据类型，强制实施强类型化。

下表列出了 System.Collections.Generic 命名空间中的一些常用类：

类 说明
Dictionary<TKey,TValue> 表示基于键进行组织的键/值对的集合。
List 表示可按索引访问的对象的列表。 提供用于对列表进行搜索、排序和修改的方法。
Queue 表示对象的先进先出 (FIFO) 集合。
SortedList<TKey,TValue> 表示基于相关的 IComparer 实现按键进行排序的键/值对的集合。
Stack 表示对象的后进先出 (LIFO) 集合。

2.1.1 Dictionary<TKey,TValue>

表示键和值的集合。

在C#中,Dictionary能提供快速的基于键值的元素查找。我们先来看一下定义

public class Dictionary<TKey,TValue> : System.Collections.Generic.ICollection<System.Collections.Generic.KeyValuePair<TKey,TValue>>, System.Collections.Generic.IDictionary<TKey,TValue>, System.Collections.Generic.IEnumerable<System.Collections.Generic.KeyValuePair<TKey,TValue>>, System.Collections.Generic.IReadOnlyCollection<System.Collections.Generic.KeyValuePair<TKey,TValue>>, System.Collections.Generic.IReadOnlyDictionary<TKey,TValue>, System.Collections.IDictionary, System.Runtime.Serialization.IDeserializationCallback, System.Runtime.Serialization.ISerializable

从定义中可以看到，Dictionary继承了很多父类。大体的功能就是让他能实现枚举，快速查找，比较，筛选等基本操作。

可详细参考官方文档：Dictionary<TKey,TValue> 类

下面列举一个简单的示例

//定义学生的学号和姓名
var stu = new Dictionary<int, string>();
//像集合中添加元素
stu.Add(1,"Zhangsan");
stu.Add(2, "Lisi");
stu.Add(3, "8yyy");
//遍历集合输出
foreach (var it in stu)
{
    Console.WriteLine($"学号:{it.Key}  姓名:{it.Value}");
}
Console.WriteLine($"集合中一共有{stu.Count}个元素");
//获取集合的类型
Console.WriteLine(stu.GetType());
//移除集合中所有的键值
stu.Clear();
Console.WriteLine($"集合中一共有{stu.Count}个元素");

2.1.3 Queue 类

队列。表示对象的先进先出集合。

详细可参考官方文档：https://learn.microsoft.com/zh-cn/dotnet/api/system.collections.generic.queue-1?view=net-7.0

下面列举一个简单的示例

Queue<int> nums = new Queue<int>();
         for (int i = 0; i < 10; i++)
         {
             nums.Enqueue(i);    //添加元素,相当于C++中的push
         }
         Console.WriteLine("原队列:");
         foreach (var item in nums)
         {
             Console.Write(item+" ");
         }
         Console.WriteLine();
         Console.WriteLine($"当前队列一共有{nums.Count}个元素"); //返回队列中的元素数,相当于C++中的size
         nums.Dequeue();         //移除元素,相当于C++中的pop
         Console.WriteLine("移除元素后的队列:");
         foreach (var item in nums)
         {
             Console.Write(item + " ");
         }
         Console.WriteLine();
         Console.WriteLine($"当前队列一共有{nums.Count}个元素");
         if (nums.Contains(8) == true)   //判断某个元素是否在队列中
         {
             Console.WriteLine($"8在这个队列中");
         }
         Console.WriteLine($"队列开始的元素为:{nums.Peek()}"); //相当于c++中的front
         int[] arr = new int[10];
         arr=nums.ToArray();			//将队列中的元素复制到arr数组中
         Console.WriteLine("数组arr中的元素:");
         foreach (var item in arr)
         {
             Console.Write(item + " ");
         }

2.1.4 SortedList<TKey,TValue>

表示键/值对的集合，这些键和值按键排序并可按照键和索引访问。

SortedList最合适对一列健/值对进行排序，在排序时，是对键进行排序，SortedList 是 Hashtable 和 Array 的混合。当使用 Item 索引器属性按照元素的键访问元素时，其行为类似于 Hashtable。当使用 GetByIndex 或 SetByIndex 按照元素的索引访问元素时，其行为类似于 Array。

SortedList 在内部维护两个数组以将数组存储到列表中；即，一个数组用于键，另一个数组用于相关联的值。每个元素都是一个可作为 DictionaryEntry 对象进行访问的键/值对。键不能为空引用（Visual Basic 中为 Nothing），但值可以。SortedList 的容量是列表可拥有的元素数。随着向 SortedList 中添加元素，容量通过重新分配按需自动增加。可通过调用 TrimToSize 或通过显式设置 Capacity 属性减少容量。SortedList 的元素将按照特定的 IComparer 实现（在创建SortedList 时指定）或按照键本身提供的 IComparable 实现并依据键来进行排序。不论在哪种情况下，SortedList 都不允许重复键。

索引顺序基于排序顺序。当添加元素时，元素将按正确的排序顺序插入 SortedList，同时索引会相应地进行调整。若移除了元素，索引也会相应地进行调整。因此，当在SortedList 中添加或移除元素时，特定键/值对的索引可能会更改。

由于要进行排序，所以在 SortedList 上操作比在 Hashtable 上操作要慢。但是，SortedList 允许通过相关联键或通过索引对值进行访问，可提供更大的灵活性。

由于 SortedList 的每个元素都是一个键/值对，因此元素类型既不是键的类型，也不是值的类型。而是 KeyValuePair<(Of <(TKey, TValue>)>) 类型。

以上内容转自：https://www.cnblogs.com/lgx5/p/6385680.html

详细可参考官方文档：https://learn.microsoft.com/zh-cn/dotnet/api/system.collections.generic.sortedlist-2?view=net-7.0

下面列举一个简单的示例

SortedList<int, string> stu = new SortedList<int, string>();
         stu.Add(1,"zhangsan");  //添加元素
         stu.Add(2,"lisi");
         stu.Add(8,"8yyy");
         foreach (KeyValuePair<int,string> item in stu)
         {
             Console.WriteLine($"键{item.Key},值{item.Value}");
         }
         Console.WriteLine($"当前SortedList中一共可以包含{stu.Capacity}个元素");
         Console.WriteLine();
         stu.Add(5, "wangwu");   //SortedList会根据键进行排序
         stu.Add(9, "zhaoyun");
         foreach (KeyValuePair<int, string> item in stu)
         {
             Console.WriteLine($"键{item.Key},值{item.Value}");
         }
         Console.WriteLine($"当前SortedList中一共可以包含{stu.Capacity}个元素");
         Console.WriteLine($"当前SortedList中一共有{stu.Count}个元素");
         if (stu.ContainsKey(8)) //确定 SortedList<TKey,TValue> 是否包含特定键。
         {
             Console.WriteLine($"当前SortedList中包含'8'键");
         }
         if (stu.ContainsValue("8yyy"))  //确定 SortedList<TKey,TValue> 是否包含特定值。
         {
             Console.WriteLine($"当前SortedList中包含'8yyy'值");
         }
         Console.WriteLine();
         Console.WriteLine("移除键为9的元素");
         stu.Remove(9);  //从 SortedList<TKey,TValue> 中移除包含指定键的元素。
         Console.WriteLine("移除所有为2的元素");
         stu.RemoveAt(2);    //移除 SortedList<TKey,TValue> 的指定索引处的元素。
         foreach (KeyValuePair<int, string> item in stu)
         {
             Console.WriteLine($"键{item.Key},值{item.Value}");
         }
         Console.WriteLine($"当前SortedList中一共可以包含{stu.Capacity}个元素");
         Console.WriteLine($"当前SortedList中一共有{stu.Count}个元素");
         Console.WriteLine();
         stu.Capacity = 4;   //获取或设置 SortedList<TKey,TValue> 可包含的元素数。
         Console.WriteLine($"当前SortedList中一共可以包含{stu.Capacity}个元素");

2.1.5 Stack 类

栈。表示相同指定类型的实例可变大小的后进先出 (LIFO) 集合。

详细可参考官方文档：https://learn.microsoft.com/zh-cn/dotnet/api/system.collections.generic.stack-1?view=net-7.0

下面列举一个简单的示例

Stack<int> st = new Stack<int>();
         for (int i = 0; i < 5; i++)
         {
             st.Push(i); //在 Stack<T> 的顶部插入一个对象。
         }
         Console.WriteLine("原始栈");
         foreach (var item in st)
         {
             Console.Write(item+" ");
         }
         Console.WriteLine();
         Console.WriteLine($"当前栈的元素个数{st.Count}");    //输出元素的个数
         Console.WriteLine("删除一个元素");
         st.Pop();   //删除并返回 Stack<T> 顶部的对象。
         Console.WriteLine($"当前栈的元素个数{st.Count}");    //输出元素的个数
         foreach (var item in st)
         {
             Console.Write(item + " ");
         }
         Console.WriteLine();
         Console.WriteLine(st.Peek());//返回顶部的元素但不删除
         if (st.Contains(3))
         {
             Console.WriteLine($"3在这个栈中");
         }
